sizefmt

This small library allows to easily format file sizes in a human-readable format.

  • SizeBase

    Declaration

    struct SizeBase(Config config);

    Wraps a size value of type which is formatted appropriately when output as text.

    • value

      Declaration

      ulong value;

      The size that should be formatted.

    • toString

      Declaration

      const void toString(scope void delegate(const(char)[]) sink, FormatSpec!char fmt);

      Formats the size according to the format fmt, automatically choosing the prefix and performing the unit conversion.

      Discussion

      The size is formatted as a floating point value, so fmt has to be a floating-point-value format specification (s, f, F, e, E, g, G, a or A). But when the unit is just bytes, it is formatted as an integer.

  • Size

    Declaration

    alias Size = SizeBase!(Config("B", "byte", "bytes", cast(PrefixUse)0, " ", false)).SizeBase;

    Size struct with default options.

    Examples

    1. assert("%s".format(Size(0)) == "0 B"); assert("%s".format(Size(1)) == "1 B"); assert("%s".format(Size(42)) == "42 B"); assert("%g".format(Size(1024)) == "1 KB"); assert("%.2f".format(Size(2_590_000)) == "2.47 MB");

  • IECSize

  • Config

    Declaration

    struct Config;

    Configuration of size format.

    Examples

    1. enum Config config = { symbol: "O", unitName: "octet", unitNamePlural: "octets", prefixUse: PrefixUse.IEC, useNameIfNoPrefix: true }; alias MySize = SizeBase!config; assert("%4.1f".format(MySize(0)) == " 0 octet"); assert("%4.1f".format(MySize(1)) == " 1 octet"); assert("%4.1f".format(MySize(42)) == " 42 octets"); assert("%4.1f".format(MySize(1024)) == " 1.0 KiO"); assert("%4.1f".format(MySize(2_590_000)) == " 2.5 MiO");

  • PrefixUse

    Declaration

    enum PrefixUse: int;

    The use of prefix when formatting long size values.

    • binary

      Declaration

      binary

      Sizes will be formatted using the traditional binary prefixes, e.g. 1024 bytes = 1 kilobyte = 1 KB.

    • IEC

      Declaration

      IEC

      Sizes will be formatted using the IEC recommandations for binary prefixes equivalent of a multiple of 1024, e.g. 1024 bytes = kibibyte = 1 KiB.

    • decimal

      Declaration

      decimal

      Sizes will be formatted using decimal prefixes, e.g. 1024 bytes = 1.024 kilobyte = 1.024 kB.
